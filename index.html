definicion de html semantico

HTML semantico significa que, aunque todo puede ser div o spans, existen etiquetas (tags) para definir semanticamente el elemento segun su contenido

esto es un head para agregar un encabezado
esto es un nav para agregar menu de navegacion
esto un aside para agregar contenido relevante lateralmente
esto es un section y agregar y agrega secciones a la pagina
